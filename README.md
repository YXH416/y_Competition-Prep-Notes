# 备赛记录

本仓库用于保存 K230/CanMV 视觉识别、MSPM0 双轴云台控制、硬件资料与测试视频。

## 内容概览

- `03.vision_01 (1).py`：K230/CanMV 靶面矩形检测与串口输出程序。程序基于第一版 Python 视觉方案继续开发，包含目标筛选、中心点滤波、丢靶保持、串口控制帧和诊断信息。
- `vision_02.py`：第二版视觉识别优化代码，加入 A4 类黑边目标评分、红色中心细化/备用检测、锁定保持及降低预览刷新频率等处理。部分运行场景可达到约 50 FPS，但当前仍存在误识别，需要继续调整筛选阈值和跟踪策略。
- `【云台】03_step_motor_已完成_可以控制速度和角度/`：MSPM0G3507 双步进电机云台工程，支持电机 1、2 初始化，以及方向、启停、速度、步数和相对角度控制；同时加入 K230 数据解析与云台闭环控制模板。
- `固件烧录教程/`：MSPM0G3507 云台测试版 TI-TXT 固件、嘉立创网页烧录教程、VS Code Serial Monitor 串口调试教程及操作截图；同时包含解决 K230 与舵机云台直接 USART 通信问题的 `K230_main_CHANGE_20260719_07.py`。MSPM0 网页烧录使用默认 9600，固件运行日志调试使用 115200。
- `测试视频/`：视觉识别与跟踪效果测试录屏。前两段测试约为 65 FPS；红线像素提取方案演示约为 30 FPS，当前为伪代码形式，仍需结合小车速度继续优化；第二版视觉代码在部分场景约为 50 FPS，但仍存在误识别。
- `k230源码.txt`、`k230示例.txt`、`云台安装教程.txt`：相关资料和参考链接。

## 当前状态

- K230 端可输出目标位置/误差信息；MSPM0 端包含 `$<length>,15,<x>,<y>,<width>,<height>,#` 数据帧解析函数。
- 云台工程已经包含电机 1、2 的初始化和控制函数调用示例。
- 闭环部分提供基于 PID 思路的调参模板；当前代码实际启用的是 P 项（`Kp`）配合低通滤波、死区、单次修正限幅和软件角度限位，`Ki`、`Kd` 尚未接入。
- 摄像头光心与激光光轴之间的物理距离尚未测量，当前距离参数为 `0.0 mm`，视差补偿未完成实物标定。

## 使用提醒

上电前应确认两个电机的轴映射、方向、机械零位和软限位。测量并填写摄像头—激光距离、偏移方向及相机内参后，才能进行实际命中精度测试。调试电机方向和机械结构时请关闭激光。
