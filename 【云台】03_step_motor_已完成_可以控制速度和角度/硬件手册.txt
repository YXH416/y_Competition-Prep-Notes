2. GPIO 数量说明
======================================================================

SysConfig 左侧显示 GPIO = 3，表示 3 个 GPIO 分组，不是 3 个引脚：

1) STEP_MOTOR
   GPIOA: PA13/DIR2、PA14/DCY2、PA15/SLP2、PA16/RST2

2) STEP_MOTOR1_A
   GPIOA: PA31/DIR1、PA7/RST1

3) STEP_MOTOR1_B
   GPIOB: PB24/SLP1、PB20/DCY1

普通数字输出 GPIO 共 8 个。PWM、UART、I2C、晶振和调试引脚属于外设复用
引脚，不计入上述 GPIO 分组数量。

6. K230 UART2配置
======================================================================

配置名: K230_UART
外设实例: UART2
外设时钟: 40,000,000 Hz
波特率: 115200
格式: 8 data bits, no parity, 1 stop bit, no flow control
TX: PA21 -> K230 RXD(IO10)
RX: PA22 <- K230 TXD(IO9)
外设中断: RX
NVIC 中断: UART2_INT_IRQn
中断函数: UART2_IRQHandler，实现在 k230.c
NVIC 开启位置: K230_Init()

K230 外部接线:
K230 5V  -> 开发板 5V/VCC
K230 GND -> 开发板 GND
K230 TXD(IO9)  -> PA22
K230 RXD(IO10) -> PA21

目标跟踪协议:
$<length>,15,<x>,<y>,<width>,<height>,#

x、y 为目标框左上角，MSPM0 计算中心坐标：
center_x = x + width / 2
center_y = y + height / 2

k230.c 的 UART2 ISR 只负责接收和组帧；K230_PollTarget() 在 main 主循环
中完成字段解析，避免在中断中执行复杂字符串转换。

10. 冲突和风险检查
======================================================================

10.1 物理引脚冲突
结果: 无冲突。
检查方法: 对 empty.syscfg 中全部 PAxx/PBxx 分配和建议分配进行分组计数，
每个物理引脚出现 1 次，重复项为 0。

10.2 外设冲突
结果: 无冲突。
TIMA1 仅用于电机1，TIMG0 仅用于电机2，UART0 仅用于 PRINT，UART2 仅
用于 K230，I2C1 仅用于 OLED。

10.3 中断冲突
结果: 无冲突。
TIMA1、TIMG0、UART0、UART2 均有独立向量和独立 ISR。

10.4 需要注意的限制

1) 0.05625 度/脉冲必须与实际电机和驱动器微步配置一致。该值通常对应
   1.8 度电机的 1/32 微步；若硬件微步模式不同，速度和角度都会产生
   固定比例误差。

2) step_set_speed() 将定时器周期限制在 800 至 65535。受 16 位周期上限
   影响，电机1理论最低脉冲频率约 152.59 Hz，最低角速度约 8.58 度/秒；
   电机2理论最低脉冲频率约 76.29 Hz，最低角速度约 4.29 度/秒。请求更低
   速度时会被周期上限钳位，实际速度会高于请求值。

3) step_set_speed() 和 step_motor_set_angle() 的参数类型都是 uint8_t，
   单次角度及速度输入范围为 0 至 255。需要大于 255 度的单次运动时应将
   API 参数扩展为 uint16_t 或 uint32_t。

4) K230 接收缓冲区最大 64 字节，只保存一帧待解析数据。main 必须持续
   调用 K230_PollTarget()；如果主循环长时间阻塞，新完成帧会被丢弃，但
   不会覆盖正在解析的帧。

5) SysConfig 生成阶段先将步进电机控制 GPIO 清零，step_motor_init() 再将
   RST、SLP、DCY 和默认 DIR 置高。这段短暂低电平会让驱动器处于复位或
   休眠状态，属于安全启动顺序。

6) 两路 PWM 的 SysConfig 默认频率不同，但 main 在启动电机前调用
   step_set_speed(60, id) 重设周期，因此不会使用默认 500/1000 Hz 运行。

7) 步进电机驱动电源 11 至 26 V 及 K230 5 V 属于外部电源连接，不由
   SysConfig 管理。所有模块必须与 MSPM0 共地，禁止将电机高压接到 MCU
   逻辑引脚。
